# SonarQube-Docker
[SonarQube](https://www.sonarqube.org/) sobre [Docker](https://www.docker.com/).

Para arrancar SonarQube, ejecutar el siguiente comando donde tengas el fichero .yml:
```
$ docker-compose up -d
```
